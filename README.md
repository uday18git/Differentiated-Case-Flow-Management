# Differentiated Case Flow Management

## Directions to use ->

### git clone
### conda create -p venv python==3.8 -y
### conda activate venv
### pip install -r requirements.txt
### python app.py

### home page -> http://127.0.0.1:5000
### predict data page  -> http://127.0.0.1:5000/predictdata
